# ETHACK
--------------

How to use

```
npm i ethack

ethack scan 100
```



Result - Make the report of serial contributors

```

| Participated  | Address       |
| ------------- | ------------- |
| 100           | 0x121....     |
| 100           | 0x122....     |
| 99            | 0x123....     |
| 97            | 0x124....     |
| 90            | 0x125....     |
```
